# Product Dashboard

Dynamic product dashboard built with React, Material UI, and CSS Modules.

## Features

- Dynamic rendering of product cards
- Availability filter for in-stock products
- Conditional rendering for no in-stock state
- Different visual styling for out-of-stock products

## Project Structure

- src/index.js
- src/App.js
- src/components/ProductList.jsx
- src/components/ProductCard.jsx
- src/styles/ProductCard.module.css

## Setup

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Run Tests

```bash
npm test
```

## Build for Production

```bash
npm run build
```
